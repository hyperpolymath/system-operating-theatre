== Technology Decisions

[NOTE]
====
The original README.md references D language implementation (`dub build`).
The Hyperpolymath Standard in CLAUDE.md strongly favors Rust for CLI/systems work.

*Decision pending* ‚Äî final implementation language to be confirmed in Phase 1 specifications.
====

=== Candidate Approaches

[cols="1,2,2"]
|===
|Option |Pros |Cons

|*Rust*
|Aligns with Hyperpolymath Standard; excellent systems tooling; memory safety
|Steeper learning curve; longer compile times

|*D*
|Fast development; good systems programming; existing README reference
|Not in Hyperpolymath "preferred" list; smaller ecosystem

|*Hybrid*
|Core in Rust, scripting glue in ReScript/Gleam
|Added complexity; multiple build systems
|===
